# Todo GUI App — 依賴說明
#
# 本專案使用 Python 標準函式庫，無需安裝第三方套件。
#
# 使用的標準庫模組：
#   - tkinter  (GUI 框架，Python 內建)
#   - tkinter.ttk (主題化 Widget)
#   - json     (資料持久化)
#   - uuid     (產生唯一 ID)
#   - datetime (時間戳記)
#   - pathlib  (跨平台路徑操作)
#   - typing   (型別提示)
#
# 執行方式：
#   python main.py
#
# Python 版本需求：>= 3.8
